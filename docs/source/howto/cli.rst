
Command-Line Interface (CLI)
============================

**geoextent** is called with this Command::
   
   python3 geoextent [-h] [-b] [-t] input

**input**
   Input file or path

**-h, --help**
   Show this help message and exit

**-b, --bounding-box**
   Extract spatial extent (bounding box)

**-t, --time-box**
   Extract temporal extent
